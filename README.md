# Парсер маршрутов для idilesom.com

Этот скрипт-парсер извлекает данные маршрутов (включая координаты треков, описания и теги) с сайта `idilesom.com` и сохраняет их в форматы JSON и CSV.

## Как начать (инструкция для новичков)

### 1. Установка зависимостей
Для работы парсера у вас должен быть установлен Node.js. 
Откройте терминал (или консоль) в папке с этим проектом и выполните команду:
```bash
npm install
```
Это установит все нужные библиотеки (axios, cheerio, dotenv, json2csv).

### 2. Настройка авторизации (Cookies)
Сайт может ограничивать доступ к данным для неавторизованных пользователей. Парсеру нужно передать вашу сессию (cookie).

1. Переименуйте файл `.env.example` в `.env` (или создайте файл `.env` рядом с `parser.js`).
2. Зайдите на сайт https://idilesom.com/ под своей учетной записью.
3. Откройте Инструменты разработчика в браузере (нажмите `F12` или `Ctrl+Shift+I`).
4. Перейдите во вкладку **Application** (или **Storage** в Firefox) -> **Cookies** -> `https://idilesom.com/`.
5. Скопируйте полное значение всех кук (или конкретной куки авторизации) и вставьте её в файл `.env`:
   ```env
   IDILESOM_COOKIE="ваша_длинная_строка_с_куками"
   ```

### 3. Запуск парсера
После того как зависимости установлены и куки вставлены в `.env`, запустите скрипт командой:
```bash
npm start
```
или:
```bash
node parser.js
```

### 4. Где лежат готовые данные?
После успешного завершения работы скрипта, в папке проекта автоматически появится папка `data/`, в которой будут находиться два файла с результатами парсинга:
* **`data/routes.json`** — Удобный формат для использования в разработках и API, где координаты сохраняют оригинальный массив Leaflet `[latitude, longitude]`.
* **`data/routes.csv`** — Табличный формат для открытия в Excel (или Google Таблицах), где вложенные массивы конвертированы в строки для удобного просмотра.

## Настройка маршрутов
В файле `parser.js` вы можете найти массив `urls`:
```javascript
const urls = [
  "https://idilesom.com/perm/places/3803"
];
```
Сюда вы можете добавить ссылки на другие треки, которые вам нужно спарсить!
